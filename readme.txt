Game Painter is Photoshop-like game making software.

The idea is to implement a set of commonplace tools that fundamentally define 2D games.

And make them easy to use.

1. Grids
2. Tile maps
3. Tile sets
4. Sprite sheets
5. Sprite animation
6. Automatic resource loading
7. Character and NPC objects
8. Text display and RPG-like dialog box support
9. Sound effects and music
10. Enemy objects
11. Keyboard, mouse and console controller support
12. Mobile screen resolution support (progressive app)
